Portfolio E-Commerce Project

Демонстрационный интернет-магазин, разработанный для портфолио

Разработчик: [Павловец Антон]

Этот проект создан для демонстрации навыков разработки современных веб-приложений с использованием Spring Boot и современных технологий.

О проекте

Описание проекта

Portfolio Shop - это полнофункциональное веб-приложение интернет-магазина, демонстрирующее мои навыки в разработке enterprise-приложений. Проект включает в себя все основные функции современного e-commerce сайта.

Цель проекта

Демонстрация практических навыков в создании коммерческих веб-приложений, работе с базами данных, реализации безопасности и создании удобного пользовательского интерфейса.

Основные возможности:
Каталог товаров

Просмотр товаров с описанием, ценами и изображениями

Корзина покупок

Добавление товаров в корзину и управление заказами

Система авторизации

Регистрация, вход и разграничение прав доступа

Админ-панель

Управление товарами и категориями для администраторов

Безопасность

Spring Security с CSRF защитой и шифрованием паролей

Адаптивный дизайн

Корректное отображение на всех устройствах

Технологический стек:
Spring Boot Spring Security Spring Data JPA Thymeleaf Bootstrap 5 PostgreSQL/H2 Maven Java 17
